# Career Development & Sustainability Hub

A functional database system built with Oracle APEX and SQL that connects 
students with career opportunities while tracking sustainability goals.

## Tech Stack
- Oracle APEX
- SQL / PL-SQL
- Oracle Database

## Pages & Features
- **Home Page** — landing page with system overview
- **Login Page** — secure user authentication
- **Opportunities** — browse available career opportunities
- **Students** — student profiles and management
- **Apply for Opportunities** — students can apply directly through the system
- **Applications** — track and manage submitted applications
- **Sustainability** — sustainability goals tracking and reporting

## About
Built as part of BSc Computer Science at University of Bedfordshire.
The system provides an end-to-end career development platform with 
a fully functional Oracle database backend, featuring real data 
relationships between students, opportunities and applications.
